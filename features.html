Prototype 1.1

Inquerex is a set of tools.

The current version is a free prototype.
It's compatible with macOS only.
It contains one tool: Search PDFs.


Search PDFs

Run sophisticated searches in selected pdfs.

  • Select pdfs anywhere on your computer.
  • Set complex search conditions with ease.
  • Browse results in an easy-to-read PDF.
  • Find things fast with color-coded results.
  • Click links to open all the right pages.

What would the full version look like?

  • Use a single window for all input/output.
  • Edit queries using a graphical interface.
  • See real-time changes in search results.
  • Search for matches in multiple languages.
  • Search for matches in multiple file types.
  • Run the app on any standard computer.
